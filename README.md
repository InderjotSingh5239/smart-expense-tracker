# Smart Expense Tracker
A modern and user-friendly web application to track daily expenses, manage monthly budgets, and visualize spending patterns.

## Features

✔ Set Monthly Budget  
✔ Add & Track Expenses  
✔ Interactive Dashboard  
✔ Category-wise Spending Graph (Chart.js)  
✔ Smart Insights & Suggestions  
✔ Download Expense Report (CSV)  
✔ Clean & Responsive UI  

## Tech Stack

- Python (Flask)
- HTML5, CSS3
- JavaScript
- Chart.js
- SQLite
- Pandas
